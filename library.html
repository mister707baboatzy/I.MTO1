What is I.MTO1?

I.MTO1 is a custom scripting language designed to simulate and teach basic programming logic. It uses a readable syntax similar to natural English and allows control over custom environments like terminals and interfaces.

How To Use I.MTO1

  • Use `Terminal.printout()` to output text.
  • Declare variables with `int`, `num`, `flag`, or `var`.
  • Use `Func:` blocks to define functions.
  • Call functions using `:.functionName();`
  • Use `if () {}` and `/!() {}` for conditionals.
